הוראות הפעלה - מערכת ניהול חנות בגדים
==========================================

1. הרצת השרת
-------------
השרת צריך לרוץ על מחשב אחד. כל הלקוחות מתחברים לאותו שרת.

א. פתח את הפרויקט ב-IntelliJ IDEA (או IDE אחר)

ב. מצא את הקובץ: src/server/ServerMain.java

ג. הרץ את הקובץ (Run -> Run 'ServerMain' או Shift+F10)

ד. אם הכל תקין, תראה הודעה: "Server listening on port 5000"

ה. השאר את החלון פתוח - השרת צריך לרוץ כל הזמן


2. הרצת לקוח (Client)
---------------------
כל משתמש שרוצה להשתמש בתוכנה צריך להריץ את הלקוח על המחשב שלו.

א. פתח את הפרויקט ב-IntelliJ IDEA

ב. מצא את הקובץ: src/gui/LoginWindow.java

ג. הרץ את הקובץ (Run -> Run 'LoginWindow' או Shift+F10)

ד. חלון התחברות ייפתח


3. חיבור מכמה מחשבים
---------------------
כדי שמספר מחשבים יתחברו לאותו שרת:

א. מצא את כתובת ה-IP של המחשב שרץ עליו השרת:
   - Windows: פתח CMD והקלד: ipconfig
   - חפש את "IPv4 Address" (למשל: 192.168.1.100)

ב. על כל מחשב אחר שרוצה להתחבר:
   - צור קובץ בשם: client.config
   - הקובץ צריך להיות בתיקיית הפרויקט (איפה שיש את src/)
   - כתוב בקובץ:
     serverHost=192.168.1.100
   (החלף 192.168.1.100 בכתובת ה-IP של המחשב עם השרת)

ג. הרץ את LoginWindow על המחשב הזה

ד. התוכנה תתחבר לשרת אוטומטית


4. דוגמה - שימוש עם 3 מחשבים
------------------------------
מחשב 1 (IP: 192.168.1.100):
- מריץ את ServerMain
- משאיר את השרת פתוח

מחשב 2 (IP: 192.168.1.101):
- יוצר client.config עם: serverHost=192.168.1.100
- מריץ את LoginWindow
- מתחבר לשרת

מחשב 3 (IP: 192.168.1.102):
- יוצר client.config עם: serverHost=192.168.1.100
- מריץ את LoginWindow
- מתחבר לאותו שרת

כל המשתמשים רואים את אותם נתונים ומתקשרים ביניהם בצ'אט!


5. משתמשים ברירת מחדל
-----------------------
כשמריצים את השרת בפעם הראשונה, נוצרים משתמשים אוטומטית:
- admin / admin (מנהל מערכת)
- superadmin / superadmin (מנהל על)

השתמש בהם להתחברות ראשונית.


6. פתרון בעיות
--------------
אם הלקוח לא מתחבר לשרת:
- ודא שהשרת רץ על המחשב השני
- ודא שכתובת ה-IP ב-client.config נכונה
- ודא ששני המחשבים באותה רשת (WiFi או Ethernet)
- בדוק שהפורט 5000 לא חסום ב-Firewall

אם יש שגיאות:
- ודא שכל הקבצים נטענו נכון
- ודא שיש תיקיית data/ עם קבצי JSON
- בדוק את ה-console לראות הודעות שגיאה


7. הערות חשובות
----------------
- השרת חייב לרוץ כל הזמן - אם סוגרים אותו, כל הלקוחות מתנתקים
- כל הלקוחות רואים את אותם נתונים (מוצרים, לקוחות, מכירות)
- הצ'אט עובד בין כל המשתמשים המחוברים
- כל שינוי במערכת מתעדכן אוטומטית אצל כל המשתמשים
